About This Project

Welcome to FlaskBlog — a full-featured blogging web application built using the Flask framework. It allows users to share their thoughts, follow other writers, and interact with posts through comments and likes.

✨ Features

  • 📝 Create, edit, and delete posts with a rich interface.
  • 👥 User authentication — sign up, log in, and manage profiles securely.
  • 📸 Profile customization with user pictures and bio.
  • ❤️ Like and comment on posts.
  • 🔔 Follow and unfollow other users to build your own community.
  • 📄 Paginated posts for smooth browsing.

🧩 Tech Stack

  • Backend: Flask, SQLAlchemy, Flask-Login, Flask-WTF
  • Frontend: Bootstrap 5, Jinja2 Templates
  • Database: SQLite (can be upgraded to PostgreSQL/MySQL)

🚀 Project Goals

This project was created to understand how a real-world web app works end-to-end: user management, routing, form handling, template rendering, and database integration. It also serves as a personal portfolio piece demonstrating web development skills with Flask.
